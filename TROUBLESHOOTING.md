# Fuzentry API Troubleshooting Guide

## Common Errors

### 401 Unauthorized
- Cause: Missing or invalid JWT token
- Fix: Obtain a fresh JWT via your Cognito flow

### 403 Forbidden
- Cause: Missing or invalid API key
- Fix: Use the API key from your Marketplace subscription email

### Metering Not Showing
- Expected: usage appears within 24 hours
- If >48 hours, contact support with your tenant ID
